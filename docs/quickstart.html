Get up and running with Goal Kit in minutes

Getting Started with Goal Kit

Goal Kit revolutionizes development with goal-first methodology ‚Äî define outcomes before implementation. This guide will help you get started with the Goal Kit framework.

Installation

The easiest way to get started is using uvx:

$ uvx --from git+https://github.com/Nom-nom-hub/goal-kit.git goal init <PROJECT_NAME>

Basic Workflow

Once installed, you can follow the goal-driven development workflow:

$ /goals        # Define project objectives
$ /clarify      # Clarify and validate goals
$ /strategize   # Develop implementation strategies  
$ /plan         # Create technical implementation plans
$ /tasks        # Generate actionable tasks
$ /implement    # Execute implementation

Example: Creating a Simple Web App

Let's walk through creating a simple web application using Goal Kit:

  1. Define Goals: Use /goals to define what you want to accomplish
  2. Clarify Requirements: Use /clarify to refine your objectives
  3. Strategy Session: Use /strategize to brainstorm implementation approaches
  4. Create Plan: Use /plan to generate a technical implementation plan
  5. Generate Tasks: Use /tasks to break down the plan into actionable items
  6. Implement: Use /implement to execute your plan
