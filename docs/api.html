Getting Started with the API

Learn how to start extending and integrating with Goal-Dev-Spec.

1

Import Required Modules

Start by importing the necessary components from the goal_cli package:

from goal_cli.goals import GoalManager
from goal_cli.specs import SpecManager
from goal_cli.analytics import PredictiveAnalyticsEngine
from goal_cli.governance import GovernanceSystem
2

Initialize Components

Create instances of the components you need to work with:

from pathlib import Path

# Initialize with your project path
project_path = Path("/path/to/your/project")
goal_manager = GoalManager(project_path)
analytics_engine = PredictiveAnalyticsEngine(project_path)
3

Develop Your Extension

Use the API to build custom functionality:

# Example: Create a goal with analytics
goal_data = {
    "title": "Implement new feature",
    "description": "Description of the new feature",
    "objectives": ["Objective 1", "Objective 2"],
    "priority": "high"
}

# Enhance with analytics
enhanced_goal = analytics_engine.enhance_goal_with_analytics(goal_data)

# Create the goal
goal_id = goal_manager.create_goal_from_data(enhanced_goal)
4

Integrate with CLI

Add your functionality to the CLI interface:

import typer
from goal_cli import app

@app.command()
def my_custom_command():
    """Custom command description"""
    # Your implementation here
    pass
