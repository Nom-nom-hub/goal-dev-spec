Installation Guide

Set up Goal Kit for your development environment

Installing Goal Kit

Goal Kit is designed to work with modern development environments. This guide will walk you through the installation process for different operating systems and development setups.

Prerequisites

Before installing Goal Kit, ensure you have the following:

  • Git (version 2.20 or higher)
  • Python 3.7+ (with pip)
  • or uv (recommended: version 0.2.0 or higher)
  • Node.js and npm (for some workflows)
  • Access to GitHub (for installation via uvx)

Recommended Installation: Using uvx

The easiest way to use Goal Kit is through uvx, which allows you to run it without permanent installation:

$ uvx --from git+https://github.com/Nom-nom-hub/goal-kit.git goal init <PROJECT_NAME>

This command will run the latest version of Goal Kit from GitHub to initialize your project. You can use this command for any Goal Kit functionality without installing it permanently.

Alternative Installation: Using uv

If you prefer to install and use Goal Kit permanently:

$ uv tool install --from git+https://github.com/Nom-nom-hub/goal-kit.git
$ goal init <PROJECT_NAME>

Development Installation

If you want to contribute to Goal Kit or modify it for your own use:

$ git clone https://github.com/Nom-nom-hub/goal-kit.git
$ cd goal-kit
$ uv sync
$ uv run goal init <PROJECT_NAME>

Verifying Installation

After installation, you can verify that Goal Kit is working correctly:

$ goal --version

If you're using uvx, you can run:

$ uvx --from git+https://github.com/Nom-nom-hub/goal-kit.git goal --version

Troubleshooting

If you encounter any issues during installation:

  • Ensure Git is properly configured and accessible from your command line
  • Check that you have internet access to reach GitHub
  • Verify that your Python installation is working correctly
  • Try running the installation command with elevated permissions if necessary

Next Steps

Now that you've installed Goal Kit, check out our Quick Start Guide to begin using the framework or our Local Development Guide to learn how to contribute to the project.
